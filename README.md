# IoT-based-Safety-Alert-System-for-Driver-Fatigue-Alcohol-Influence-Smoke-Detection-final-year
An IoT-based driver safety alert system that detects driver fatigue, alcohol influence, and smoke using sensors with ESP8266. The system sends real-time alerts through the Blynk app and activates a buzzer and vibration motor to warn the driver, helping prevent accidents and improving road safety.
